## Features  

### Daily Journal  
- Write text entries  
- Add photos from gallery or camera  

### Vision Board  
- Upload and arrange images, quotes, and affirmations in a collage style  

### Habit Tracker  
- Track daily habits with icons  
- Build streaks  

### Minimal Aesthetic  
- Pastel color palette  
- Smooth transitions  
- Elegant typography
